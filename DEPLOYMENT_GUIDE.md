# 📱 应用部署完整指南

你现在有了一个完整的 PWA 应用，有 **3 种使用方式**。

## 🎯 三种使用方式对比

### 方式 1️⃣：本地网络（在家使用）

```
电脑 ←WiFi→ 手机
```

**优势：**
- ✅ 无需账号
- ✅ 最快速度
- ✅ 完全离线可用
- ✅ 无成本

**步骤：**
1. 电脑启动: `python -m http.server 8000`
2. 手机输入: `http://[电脑IP]:8000`
3. 安装应用

**限制：**
- ❌ 需要在同一网络
- ❌ 电脑必须开机

**文档：** 📄 `MOBILE_INSTALL.md`

---

### 方式 2️⃣：云部署（任何地方使用）⭐ **推荐**

```
手机 ←互联网→ 云服务器 (Vercel)
```

**优势：**
- ✅ 全球任何地方可访问
- ✅ 脱离电脑使用
- ✅ 24/7 在线
- ✅ 完全免费
- ✅ 完全离线可用
- ✅ 可分享给朋友

**步骤：**
1. 创建 GitHub Token
2. 运行 `deploy.bat` 上传代码
3. 在 Vercel 部署
4. 在手机上安装应用

**成本：**
- 🆓 完全免费

**文档：** 📄 `CLOUD_DEPLOY.md` 或 `QUICK_DEPLOY.md`

---

### 方式 3️⃣：自部署（自己的服务器）

```
手机 ←互联网→ 你的服务器
```

**优势：**
- ✅ 完全控制
- ✅ 无需依赖第三方
- ✅ 可添加后端服务

**成本：**
- 💰 需要购买服务器（约 $5-20/月）

**不推荐初学者使用**
