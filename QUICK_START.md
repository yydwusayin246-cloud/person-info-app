## 🌍 部署方案（选择其一）

### 方案 A：🇨🇳 **中国用户推荐 - Netlify 部署**（5分钟）

**为什么选择 Netlify？** 
- 在中国大陆有 CDN 节点
- 第一次访问不需要 VPN ✓
- 完全免费

**步骤：**
1. 访问 https://github.com/new 创建新仓库
   - 仓库名：`person-info-app`
   - 选择"Public"

2. 克隆到电脑并上传文件：
   ```bash
   git clone https://github.com/你的用户名/person-info-app.git
   cd person-info-app
   # 把 person_info_app 文件夹中的所有文件复制到这里
   git add .
   git commit -m "Initial commit"
   git push
   ```

3. 访问 https://netlify.com
   - 用 GitHub 账号登录
   - 点击"New site from Git"
   - 选择你的 person-info-app 仓库
   - 保持默认设置，点击"Deploy"

4. 部署完成后，你会获得一个链接（如 `https://xxx.netlify.app`）
   
5. **在手机上访问这个链接**，然后添加到主屏幕即可

---

### 方案 B：**国际用户 - Vercel 部署**

类似 Netlify，但 Vercel 在中国大陆可能较慢。步骤相同，只需在 https://vercel.com 而不是 netlify.com 操作。

---

### 方案 C：**完全本地 - 无需云服务**

在你的局域网内部署，电脑和手机同一 WiFi：

```bash
# 在项目文件夹运行
python server.py
# 或者
python -m http.server 8000

# 手机访问：http://你的电脑IP:8000
# （找电脑IP：Windows CMD 输入 ipconfig）
```

## ⚠️ 常见问题

**Q：为什么我的手机上看不到三个界面分开？**
- A：需要在手机上测试，不要用电脑浏览器的手机模拟（模拟效果不完全）

**Q：第一次打开需要多长时间？**
- A：首次打开需要下载缓存（几秒钟），之后完全离线可用

**Q：我的数据安全吗？**
- A：数据全部存储在你的手机本地，不上传到任何服务器

**Q：能在多个手机上同步数据吗？**
- A：暂时不支持（需要额外的后端数据库），每个手机是独立的
