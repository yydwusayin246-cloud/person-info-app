# 📱 手机安装指南 - PWA 版本

你的应用已升级为 **PWA（Progressive Web App）**！现在可以在手机上像原生应用一样安装和使用。

## ✨ PWA 的优势

- 📱 **安装到主屏** - 像真正的应用一样
- 📴 **完全离线工作** - 无需网络连接
- 💾 **本地数据存储** - 数据保存在手机中
- ⚡ **快速启动** - 像原生应用一样快
- 🔄 **自动更新** - 后台自动检查更新
- 🚀 **无需 App Store** - 直接安装使用

---

## 🚀 安装步骤

### 📲 Android 手机 (Chrome/Edge)

#### **方式 1：自动安装提示（推荐）**

1. **连接到网络** - 确保手机连接到 WiFi 或数据
2. **打开应用**：
   - 电脑：启动 Python 服务器（运行 `python -m http.server 8000`）
   - 手机：在浏览器中输入 `http://[你的电脑IP]:8000`
3. **等待安装提示** - 浏览器会自动显示"⬇️ 安装应用"按钮
4. **点击安装** - 点击"⬇️ 安装应用"按钮
5. **确认安装** - 弹出窗口中点击"安装"
6. ✅ **完成** - 应用会添加到主屏幕

#### **方式 2：菜单安装**

1. 打开应用
2. 点击浏览器菜单（⋮ 三个点）
3. 选择"⬇️ 安装应用"
4. 确认安装
5. ✅ 完成

### 🍎 iPhone/iPad (Safari)

#### **步骤**

1. **打开应用**：在 Safari 中访问应用 URL
2. **点击分享按钮** - 点击底部的"分享"按钮（⬆️）
3. **向下滑动** - 向下滑动菜单
4. **选择"添加到主屏幕"** - 点击此选项
5. **确认名称** - 可以修改应用名称（保持默认即可）
6. **点击"添加"** - 确认安装
7. ✅ **完成** - 应用会添加到主屏幕

## 🖥️ 电脑端准备

### 获取电脑 IP 地址

#### **Windows**
```powershell
# 打开命令提示符或 PowerShell，运行：
ipconfig

# 查找 "IPv4 地址"，类似于：192.168.1.100
```

#### **Mac/Linux**
```bash
ifconfig
# 或
ip addr show
```

### 启动服务器

```bash
# 进入项目目录
cd d:\YYD\copilot\person_info_app

# 启动 Python 服务器
python -m http.server 8000

# 输出类似：
# Serving HTTP on 0.0.0.0 port 8000
```

### 确保防火墙允许

- Windows：可能需要允许 Python 通过防火墙
- 确保电脑和手机在同一个网络上

## 🔗 在手机上访问

### 第一次访问

1. **获取电脑 IP** - 例如：`192.168.1.100`
2. **打开手机浏览器** - Chrome、Safari、Edge 或 Firefox
3. **输入地址**：`http://192.168.1.100:8000`
4. **等待加载** - 应用会加载

### 之后的使用

1. **直接从主屏幕打开** - 就像原生应用一样
2. **无需网络** - 一旦安装，即使断网也能继续使用
3. **数据同步** - 所有数据保存在手机本地

## ❓ 常见问题

### Q: 为什么在手机上看不到"安装应用"提示？
**A:** 可能的原因：
- 浏览器不支持 PWA（使用 Chrome、Edge、Safari 或 Firefox）
- 应用加载不完整（刷新页面）
- 已经安装过了
- 尝试手动安装（见上面的"菜单安装"）

### Q: 应用离线能使用吗？
**A:** 是的！完全离线可用，因为：
- 所有代码都已缓存到手机
- 数据保存在本地 localStorage
- 无需网络连接

### Q: 如何更新应用？
**A:** 自动更新：
- 应用会在后台检查更新
- 有新版本时会提示
- 刷新页面立即使用新版本

### Q: 支持哪些浏览器？
**A:** PWA 支持：
- ✅ Chrome/Chromium 80+
- ✅ Safari 14.1+ (iPhone/iPad)
- ✅ Edge 79+
- ✅ Firefox (部分支持)
- ✅ Samsung Internet

### Q: 数据会丢失吗？
**A:** 通常不会，但以下情况会丢失：
- 清除浏览器缓存
- 卸载应用
- 用户手动删除
- **建议**：定期导出备份

### Q: 能在其他设备上使用吗？
**A:** 是的：
- 每个设备独立安装
- 每个设备有独立的数据
- 可以同步服务器（需要后端支持）

## 🎯 最佳实践

1. **定期检查更新** - 应用会自动检查，但可以手动刷新
