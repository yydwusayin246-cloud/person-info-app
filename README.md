# 👥 人员信息管理系统

一个功能完整的人员信息管理应用，支持添加、查看和搜索人员信息。

## 📋 功能特性

### 1️⃣ 输入信息界面
- 📝 输入人员基本信息
- 🔒 必填字段验证
- 💾 自动保存到本地存储
- 🔄 快速清空表单

**输入字段：**
- 姓名 *（必填）
- 店铺挡位
- 行业熟练程度（初级/中级/高级/专家）
- 年龄
- 性格特点
- 意愿程度（0-100%）
- 微信名称

### 2️⃣ 信息总览界面
- 📊 Excel表格式展示所有人员数据
- 👥 显示总人数统计
- ✏️ 编辑现有记录
- 🗑️ 删除记录
- 📱 响应式表格设计

### 3️⃣ 搜索功能
- 🔍 按姓名搜索
- 🎂 按年龄搜索
- 🏢 按店铺挡位搜索
- 📚 按行业熟练程度搜索
- 🔄 快速清空搜索条件
- 📋 显示搜索结果统计

## 🚀 快速开始

### 前置要求
- Python 3.6+
- 任何现代浏览器（Chrome、Firefox、Safari、Edge 等）

### 运行应用

**方法 1：使用 Python 服务器（推荐）**

```bash
# 进入项目目录
cd person_info_app

# 运行服务器
python server.py
```

应用会自动在浏览器中打开 `http://localhost:8000`

**方法 2：直接打开 HTML 文件**

直接在浏览器中打开 `index.html` 文件

**方法 3：使用 Python 内置服务器**

```bash
cd person_info_app
python -m http.server 8000
```

然后在浏览器中访问 `http://localhost:8000`

## 📱 在移动设备上使用

### 手机访问本地服务
1. 确保手机和电脑连接到同一网络
2. 找到你的电脑 IP 地址（Windows 中运行 `ipconfig`）
3. 在手机浏览器中访问 `http://<你的电脑IP>:8000`

示例：如果 IP 是 192.168.1.100，访问 `http://192.168.1.100:8000`

## 💾 数据存储

- 所有数据保存在浏览器的 **localStorage** 中
- 数据在浏览器关闭后仍然保留
- 清除浏览器缓存会删除所有数据
- 建议定期导出数据作为备份

## 🐛 故障排除

**问题：页面无法加载**
- 确保 server.py 正在运行
- 检查是否在正确的目录中
- 尝试访问 `http://localhost:8000`

**问题：数据丢失**
- 不要清除浏览器缓存
- 定期备份数据
- 考虑在多个浏览器中保存副本

**问题：在手机上无法连接**
- 确保手机和电脑在同一网络
- 确保防火墙未阻止端口 8000
- 使用正确的 IP 地址
